 API REST de Usuarios - Flask + SQLite
 Descripción

Este proyecto consiste en el desarrollo de una API REST para la gestión de usuarios, implementada en Python utilizando Flask y una base de datos SQLite.

La API permite realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) mediante el uso de métodos HTTP estándar, y fue diseñada para ser probada y validada con herramientas como Postman.

 Tecnologías utilizadas
Python 3
Flask
SQLite
Postman (para pruebas)
